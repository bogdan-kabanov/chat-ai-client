# ChatGPT Frontend

React + TypeScript + Vite + MUI — фронтенд для ChatGPT веб-приложения.

## Стек

- **React 18** — UI-библиотека
- **TypeScript** — типизация
- **Vite** — сборщик
- **MUI (Material UI)** — компоненты и иконки
- **Pug** — шаблонизатор (опционально)
- **Web Speech API** — голосовой ввод

## Функциональность

- Поле ввода текста с отправкой на сервер
- Отображение ответов ChatGPT в виде чата
- Кнопка микрофона для голосового ввода (Web Speech API)
- Индикатор загрузки при ожидании ответа
- Обработка ошибок с уведомлениями
- Адаптивный минималистичный дизайн

## Установка и запуск

```bash
# Установить зависимости
npm install

# Запустить dev-сервер
npm run dev
```

Приложение будет доступно на `http://localhost:5173`

## Сборка

```bash
npm run build
npm run preview
```

## Конфигурация

Vite проксирует запросы `/api` на backend (`http://localhost:3001`).
Убедитесь, что backend запущен перед использованием.
